本项目定位是寻道大千游戏的辅助工具

#需求一

灵兽是寻道大千中玩家的宠物，可以参与战斗并为玩家提供加成

灵兽初始词条的槽位有四个，每个词条都可以为玩家提供属性的加成

词条的类型有 22 种，包括：
攻击、防御、生命、敏捷
击晕、暴击、连击、闪避、反击、吸血
抵抗击晕、抵抗暴击、抵抗连击、抵抗闪避、抵抗反击、抵抗吸血
强化灵兽、弱化灵兽
强化治疗、弱化治疗
强化暴伤、弱化暴伤
不同类型词条出现的概率一样

词条的品质有 3 种，包括：金、紫、蓝，其中金色品质最高，紫色次之，蓝色最低，品质高的出现概率更低，具体来说：
金色词条的出现概率为 3.2%
紫色词条的出现概率为 1%
蓝色词条的出现概率为 0.5%

因此，词条的类型加品质，一共有 66 种可能性，比如金色的攻击，紫色的强化治疗等

洗炼词条功能规则如下：
1. 四个槽位同时参与洗炼
2. 四个槽位不会出现一样的词条

#需求二

增加灵兽洗炼模拟时的词条锁定功能

锁定词条：每个词条下增加一个按钮，可以加锁和解锁，锁定时该词条在洗炼过程中不会变更
锁定词条材料消耗：锁定一个词条消耗 20 个不化岩；锁定两个词条消耗 40 个不化岩；锁定三个词条消耗 100 个不化岩；锁定四个词条没有意义
洗练材料消耗：无论是否锁定，每次洗炼消耗 20 个洗炼石

在洗炼结果下增加 UI 显示需要洗炼消耗材料的数量

注意：锁定的词条洗炼不会变动，其他变化的词条也必须遵循“四个槽位不会出现一样的词条”这个原则

#需求六

增加一个页面，用于计算灵兽洗炼到指定词条的概率

具体功能如下：
1. 玩家需要指定四个初始的词条，可以锁定其中一到三个词条
2. 除了锁定的以外，玩家可以指定任意数量的目标词条
3. 计算从初始词条变更为目标词条的概率

样例：
玩家四个初始词条是“金的强化治疗”，“金的击晕”，“蓝的攻击”，“蓝的闪避”，其中“金的强化治疗”被锁定；
除了被锁定的“金的强化治疗”，玩家指定了两个目标词条“紫的强化治疗”，“金的弱化治疗”
目的是计算出，锁定“金的强化治疗”后，剩下三个词条中，任意两个词条出现“紫的强化治疗”，“金的弱化治疗”的概率有多少
概率作为结果显示在 UI 中，以百分比呈现，精确度为两位小数，比如 3.12%

#需求七

增加一个页面，指定初始词条和目标词条后，计算最佳洗炼路径

样例：
初始词条：金的强化治疗
目标词条：金的强化治疗，蓝的强化治疗，金的弱化治疗，紫的弱化治疗
可能的洗炼路径：锁定金的强化治疗，洗炼出金的弱化治疗为止；再锁定金的强化治疗和金的弱化治疗，洗炼出紫的弱化治疗为止；最后锁定金的强化治疗、金的弱化治疗、紫的强化治疗，洗出蓝的强化治疗

上面洗炼路径是基于先用较少的材料洗概率低的，再用较多的材料来洗概率高的，从而得到最优的结果，但这个算法并不一定是最优解法
请按数学公式计算或者模拟计算得出最优的洗炼方式（用的材料最少），并用适当的 UI 进行呈现
